CSS Viewer Extension

CSSViewer is a simple CSS property viewer for Google chrome originally made by Nicolas Huon as a FireFox addon.

CSSViewer is now an open source project and now available at https://github.com/cssviewer/cssviewer. Feel free to join and to submit new ideas and feedback!

How to use :

CSSViewer shows the css parameters of any element in a web page, to enable/disable it simply click the toolbar icon and then hover any element on you want to inspect in the page.

How to uninstall :

 - Click the wrench icon on the browser toolbar.
 - Click Tools.
 - Select Extensions.
 - Find the CSSViewer extension then click Uninstall.



CSSViewer ported to Google chrome on March 2010. All credit goes to the original developer.
